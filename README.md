# sc-reactjs-routing

<p>This repository is a simple React navbar made with Style Components, react-router-dom,
react-helmet, and font-awesome icon. The navbar is styled with styled-components, uses
react-router-dom for routing between pages, and react-helmet managing for meta-data. 
It also uses font-awesome for menu icon.</p>

## Features
- Responsive design
- Styled with styled-components
- Routing with react-router-dom
- Manages meta-data with react-helmet
- Menu button icon with font-awesome

### To install dependencies
```shell
npm install
```
or
```shell
yarn or yarn install
```
### To run the app
```shell
npm run dev
```
or

```shell
yarn dev
```

### Build the app
```shell
npm run build
```
or

```shell
yarn build
```
